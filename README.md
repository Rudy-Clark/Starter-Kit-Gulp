# Starter-Kit-Gulp
gulp configs

```shell
git clone https://github.com/Rudy-Clark/Starter-Kit-Gulp
cd Starter-Kit-Gulp
npm install
```
Commands 

Clean, build, minimize and serving `localhost:8080`
```shell
gulp default 
```
or just
```shell
gulp
```
Build
```shell
gulp build
```
Clean `./build` dir
```shell
gulp clean
```
