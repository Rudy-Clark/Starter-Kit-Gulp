# Starter-Kit-Gulp
gulp configs

```shell
git clone https://github.com/Rudy-Clark/Starter-Kit-Gulp
cd Starter-Kit-Gulp
npm install 
// or
yarn install
``` 

### Commands 

Clean, build and serving `localhost:8080`
```shell
npm start
// or
yarn start
```

Build
```shell
npm build
// or
yarn build
```

Build for production
```bash
npm build:prod
// or
yarn build:prod
```
